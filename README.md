# 📌 Project Overview

This repository contains a **complete learning series of NumPy in Python**.

NumPy is one of the most important libraries for **numerical computing, data analysis, and machine learning**.

The tutorials in this project cover:

- NumPy arrays
- Array indexing and slicing
- Mathematical operations
- Random number generation
- Probability distributions
- Statistical operations
- Linear algebra utilities

The goal of this repository is to provide a **step-by-step practical learning resource for NumPy**.

---

# 🧠 How It Works

Each Python file in this repository demonstrates a **specific NumPy concept** with practical examples.

The tutorials progress from **basic to advanced topics**, helping learners understand NumPy systematically.

Learning Flow:

1. NumPy basics and array creation
2. Array indexing and slicing
3. Array operations and iteration
4. Sorting, filtering, and reshaping
5. Random number generation
6. Statistical and probability distributions
7. Mathematical and trigonometric functions

# ⚙ Installation

## 🪟 Windows

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/numpy-python-tutorials.git
```

### 2. Go to the project folder

```bash
cd numpy-python-tutorials
```

### 3. Install required packages

```bash
pip install numpy
```

### 4. Run the example file

```bash
python 1_numpyintro.py
```

---

## 🐧 Linux

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/numpy-python-tutorials.git
```

### 2. Go to the project folder

```bash
cd numpy-python-tutorials
```

### 3. Install dependencies

```bash
pip3 install numpy
```

### 4. Run the example file

```bash
python3 1_numpyintro.py
```
